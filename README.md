# Jeremiah Ogunturin — Portfolio

My personal portfolio website

Self taught using HTML, CSS, and JavaScript.


---

## About

A terminal-style portfolio site presenting my engineering background, work experience, projects, and music taste.

Features include:
- Tab-based navigation across Profile, Experience, Projects, and Music sections
- Dynamic music player with embedded Spotify previews for my ten favourite albums
- Real-time UTC clock

## Running Locally

No build step required. Clone the repo and open `index.html` in a browser:

```bash
git clone https://github.com/jeremiiiah/YOUR-REPO-NAME.git
cd YOUR-REPO-NAME
# open index.html in your browser
```

---

## Custom Domain

This site is intended to be served via a custom domain. Once purchased, configure it under **Settings → Pages → Custom domain** in the GitHub repository, and point your domain's DNS to GitHub Pages.
